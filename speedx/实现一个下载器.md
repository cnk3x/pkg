实现一个下载器

## 传入参数

1. 下载链接 URL
2. 保存目录
3. 保存文件名(可选)
4. 最大下载线程数
5. 最大分块大小
6. 下载进度，速度报告回调
7. 最大出错重试次数
8. 重试间隔时间
9. 块出错是否继续下载其他的块

## 下载步骤:

1. 检查 URL
   1. 是否支持断点续传
   2. 如果参数没有传入文件名，通过 HTTP Header ContentType 猜测文件名，如果没有通过 URL 来猜测文件名
2. 如果支持断点续传，启动多线程下载，否则单线程下载
3. 下载未完成前，在保存文件名后添加.downloading 后缀的临时文件，表示还未下载完
4. 未下载完时在块写入完成时将已下载好的信息保存在 保存文件名.downloading.info 文件中, 记录已经下载并写入完成的块。
5. 块下载失败按照传入参数配置重试
